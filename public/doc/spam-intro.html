Switch Ports Activity Monitor

Documentation

Switch Ports Activity Monitor is an in-house developed tool for monitoring switched ethernet network operations and maintenance. It consists from a script written in Perl that communicates with ethernet switches over SNMP, backend database that stores retrieved information and a WWW administration tool.
