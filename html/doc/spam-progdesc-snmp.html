Module SPAM_SNMP.pm

Description

This library contains collection of function for getting and processing information via SNMP used by SPAM collector.

Functions Overview
snmp_getif ( host, community, field ) This function retrieves given SNMP variable for all existing interfaces.
snmp_getif_cat ( host, community, field ) This function retrieves given SNMP Catalyst-specific variable for all existing interfaces.
snmp_reindex_cat ( ifdata, cross_idx ) This function reindexes hash retrieved by snmp_getif_cat() to standard ifIndex.
snmp_merge_by_ifindex ( hashes ) This function merges multiple snmp_getif() hashes into single one.
snmp_cat6k_modinfo ( host, community ) This function retrieves information about all modules installed in a Catalyst 6000-class switch.
snmp_cat6k_vlan_name ( vlan_no, host, community ) This function gets VLAN name for given VLAN number.
snmp_cdp_cache ( host, community ) This function retrieves content of CDP cache into memory.
snmp_get_syslocation ( host, community ) This function retrieves SNMP sysLocation.0 variable.
snmp_get_sysobjid ( host, community ) This function retrieves SNMP sysObjId.0 variable.
snmp_get_stp_root_port ( host, community ) This function figures out STP root port's ifindex.
snmp_get_arptable ( arpdef ) Retrieves ARP cache from defined routers.
snmp_get_mactable ( host, community, vlanlist, cdp_cache ) Retrieves bridging table from given switch.
snmp_get_vtp_info ( host, community ) Get VTP domain name and VTP mode from a switch.
snmp_vlanlist ( host, community ) Retrieves list of VLANs from a Catalyst 6000-class switch. The result is returned as a reference to VLAN number to VLAN name hash.
