Regular expressions quick reference

Character classes

elementdescription
.Match any character
[xyz]Match any of the specified characters
\sMatch white space
\SMatch non-white space
\dMatch digit
\DMatch non-digit
\wMatch alphanumeric character
\WMatch non-alphanumeric character

Quantifiers

elementdescription
*Match zero or more occurences of preceding character
+Match one or more occurences of preceding character
?Match zero or one occurence of preceding character
{x}Match exactly x occurences of preceding character
{x,y}Match between x and y occurences of preceding character
{,x}Match between zero and x occurences of preceding character
{x,}Match x or more occurences of preceding character

Positions

elementdescription
^Match beginning of the string
$Match end of the string
